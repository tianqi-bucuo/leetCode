给定一个字符串，请你找出其中不含有重复字符的 最长子串 的长度。

***

```Java
public int lengthOfLongestSubstring(String s) {
    Set<Character> set = new HashSet<>();
    int i = 0, j = 0;
    int res = 0;
    for(; i < s.length(); i++){
        while(j < s.length() && !set.contains(s.charAt(j))){
            set.add(s.charAt(j));
            res = Math.max(j - i + 1, res);
            j++;
        }
        set.remove(s.charAt(i));
    }
    return res;
}
```
