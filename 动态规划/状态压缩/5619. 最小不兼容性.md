给你一个整数数组 nums​​​ 和一个整数 k 。你需要将这个数组划分到 k 个相同大小的子集中，使得同一个子集里面没有两个相同的元素。

一个子集的 不兼容性 是该子集里面最大值和最小值的差。

请你返回将数组分成 k 个子集后，各子集 不兼容性 的 和 的 最小值 ，如果无法分成分成 k 个子集，返回 -1 。

子集的定义是数组中一些数字的集合，对数字顺序没有要求。

***

```Java
public int minimumIncompatibility(int[] nums, int k) {
    int n = nums.length, per = nums.length / k;
    Arrays.sort(nums);
    if (k == 1) {
        for (int i = 1; i < n; i++) {
            if (nums[i] == nums[i - 1]) {
                return -1;
            }
        }
        return nums[n - 1] - nums[0];
    }
    
    if (k == n) {
        return 0;
    }

    int MAX_VALUE = (nums[n - 1] - nums[0]) * k;

    int state = 1 << n;
    int[][] dp = new int[state][n];
    int[] counts = new int[state];
    for (int i = 0; i < state; i++) {
        int count = 0;
        for (int j = 0; j < n; j++) {
            dp[i][j] = MAX_VALUE;
            if ((i & (1 << j)) != 0) {
                count++;
            }
        }
        counts[i] = count;
    }
    Arrays.fill(dp[0], 0);
    for (int mask = 1; mask < state; mask++) {
        if (counts[mask] % per == 0) {
            for (int firstIndex = 0; firstIndex < n; firstIndex++) {
                if ((mask & (1 << firstIndex)) != 0) {
                    dp[mask][0] = Math.min(dp[mask][0], dp[mask ^ (1 << firstIndex)][firstIndex]);
                }
            }
            for (int pre = 1; pre < n; pre++) {
                dp[mask][pre] = dp[mask][0];
            }
        } else {
            for (int pre = 0; pre < n - 1; pre++) {
                for (int p = pre + 1; p < n; p++) {
                    if ((mask & (1 << p)) != 0 && nums[p] != nums[pre]) {
                        dp[mask][pre] = Math.min(dp[mask][pre], dp[mask ^ (1 << p)][p] + nums[p] - nums[pre]);
                    }
                }
            }
        }
    }
    if (dp[state - 1][0] >= MAX_VALUE) {
        return -1;
    }
    return dp[state - 1][0];
}
```
