给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

```Java
public void moveZeroes(int[] nums) {
	int zeroIndex = 0;
	// 遇到0跳过，可以用zero记录0的位置
	for (int i = 0; i < nums.length; i++) {
		if (nums[i] != 0) {
			nums[zeroIndex++] = nums[i];
		}
	}
	// 将后面的填充为0
	for (int i = zeroIndex; i < nums.length; i++) {
		nums[i] = 0;
	}
}
```
