在数组中的两个数字，如果前面一个数字大于后面的数字，则这两个数字组成一个逆序对。输入一个数组，求出这个数组中的逆序对的总数。


在归并排序的过程中统计逆序对

```Java
class Solution {

    public int reversePairs(int[] nums) {
        if(nums.length < 2) return 0;
        return merge(nums, 0, nums.length-1);
    }

    private int merge(int[] nums, int start, int end){
        if(start == end) return 0;
        int mid = start + (end-start) / 2;
        int[] temp = new int[end-start+1];
        int count = merge(nums, start, mid) + merge(nums, mid+1, end);
        int i = start, j = mid+1, k = 0;
        while(i <= mid && j <=end){
            // 可以看成两个从大到小的有序数组，左边数组nums[i]~nums[mid]；
            // 右边数组nums[j]~nums[end]，进行合并；
            // (重点！)如果nums[i] > nums[j]，则nums[i]可以和nums[j]及之后的所有数组成
            // 逆序对（因为都是从大到小排列），个数为：end-j+1；
            count += nums[i] > nums[j] ? end-j+1 : 0;
            temp[k++] = nums[i] > nums[j] ? nums[i++] : nums[j++];
        }
        // 上面循环结束，nums前半部分数组已经不存在大于后半部分，正常合并，不用计数；
        while(i <= mid)
            temp[k++] = nums[i++];
        while(j <= end)
            temp[k++] = nums[j++];
        for(int m = 0; m < temp.length; m ++)
            nums[start+m] = temp[m];
        return count;
    }
}
```
