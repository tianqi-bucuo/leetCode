给定两个大小为 m 和 n 的有序数组 nums1 和 nums2。  

请你找出这两个有序数组的中位数，并且要求算法的时间复杂度为 O(log(m + n))。  

你可以假设 nums1 和 nums2 不会同时为空。  

```Java
class Solution {
    public double findMedianSortedArrays(int[] nums1, int[] nums2) {
        int[] nums = new int[nums1.length + nums2.length];
        int i = 0, j = 0, k = 0;
        //由于是求中位数,只需要排序到一半的长度
        while (k <= nums.length / 2 && i != nums1.length && j != nums2.length) {
            if (nums1[i] <= nums2[j]) {
                nums[k] = nums1[i];
                i++;
            } else {
                nums[k] = nums2[j];
                j++;
            }
            k++;
        }
        if (k <= nums.length / 2 && i == nums1.length) {
            while (k <= nums.length / 2) {
                nums[k] = nums2[j];
                k++;
                j++;
            }
        } else if (k <= nums.length / 2 && j == nums2.length) {
            while (k <= nums.length / 2) {
                nums[k] = nums1[i];
                k++;
                i++;
            }
        }
        return nums.length % 2 == 0 ? (double) (nums[k-1]+nums[k-2])/2:(double) nums[k-1]; 
    }
}
```
