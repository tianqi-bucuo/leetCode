给定一个字符串，请你找出其中不含有重复字符的 最长子串 的长度。

```Java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        int i = 0,j = 1,len = 1;
        if (s.equals("")){
            return 0;
        }else if (s.length() == 1){
            return 1;
        }
        while(j < s.length()){
            if(!s.substring(i,j).contains(Character.toString(s.charAt(j)))){
                j++;
            }else {
                i = s.substring(i,j).indexOf(s.charAt(j))+i+1;
                j++;
            }
            if (j-i > len){
                len = j -i;
            }
        }
        return len;
    }
}
```
