给定一个二叉树，返回它的 后序 遍历。

```Java
class Solution {
    public void func2(TreeNode root, List < Integer > res) {
        if(root != null){
            func2(root.left, res);           
            func2(root.right, res);
            res.add(root.val);
        }
    }
    public List<Integer> postorderTraversal(TreeNode root) {
        List < Integer > res = new ArrayList < > ();
        func2(root, res);
        return res;
    }
}
```
