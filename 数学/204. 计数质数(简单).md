统计所有小于非负整数 n 的质数的数量。

***

```Java
public int countPrimes(int n) {
    int res = 0;
    boolean[] flag = new boolean[n + 1];
    for (int i = 2; i < n; i++) {
        if (!flag[i]) {
            res++;
        }
        for (int j = 2; i * j < n; j++) {
            flag[i * j] = true;
        }
    }
    return res;
}
```
